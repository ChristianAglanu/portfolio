1. Project Title

Implementation of a Centralized Security Monitoring Solution Using Wazuh

2. Objective

To implement a centralized security monitoring solution that collects, analyzes, and generates alerts from logs across multiple systems in real time, enhancing the organization’s ability to detect, investigate, and respond to cyber threats effectively.

3. Project Overview

This project focuses on deploying a Security Information and Event Management (SIEM) system using Wazuh on a dedicated infrastructure.

 

 

 

The solution enables:

  • Centralized log collection
  • Real-time threat detection
  • Security event correlation
  • Incident response support

The system is deployed on a dedicated HP PC running Ubuntu Server (TTY-only environment) to ensure stability, performance, and reduced attack surface.

4. System Architecture

4.1 Components

  • Wazuh Manager (Central server)
  • Wazuh Agents (Installed on monitored endpoints)
  • Elastic Stack / Wazuh Dashboard
  • Ubuntu Server (TTY-only)

8. Detection & Alerting

Wazuh analyzes logs using predefined rules to detect:

  • Brute force attacks
  • Unauthorized access attempts
  • Malware indicators
  • Suspicious system activity

9. Testing & Validation

Test Scenarios Performed

  • Failed login attempts (Brute force simulation)
  • Unauthorized file access
  • Suspicious command execution

Results

  • Alerts were generated successfully
  • Logs were collected in real time
  • System responded as expected

10. Challenges Encountered

  • Working in a TTY-only environment required strong command-line skills
  • Network configuration issues during setup
  • Agent connectivity troubleshooting

11. Solutions Implemented

  • Used SSH for remote management
  • Verified firewall rules and open ports
  • Ensured correct agent configuration (IP and keys)

12. Security Considerations

  • Dedicated server reduces attack surface
  • TTY-only Ubuntu minimizes unnecessary services
  • Secure communication between agents and server

13. Key Achievements

  • Successfully deployed a centralized SIEM system
  • Achieved real-time log monitoring and alerting
  • Improved visibility into system activities
  • Built a scalable security monitoring foundation

14. Conclusion

 

The implementation of Wazuh on a dedicated Ubuntu Server (TTY-only) system provides a robust and efficient centralized security monitoring solution. It enhances the organization’s capability to detect and respond to cyber threats in real time.
